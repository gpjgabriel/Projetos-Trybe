# Project Shopping Cart

# Contexto
Projeto 7 - Callback, API e Test Assíncrono

## Técnologias usadas

Front-end:
> Desenvolvido usando: CSS3, HTML5, ES6, Jest

## Instalando Dependências

> Frontend
```bash
cd src/
npm install
``` 
## Executando aplicação

* Para rodar o front-end:

  ```
    cd src/ && npm start
  ```

## Executando Testes

* Para rodar todos os testes:

  ```
    npm test
  ```
