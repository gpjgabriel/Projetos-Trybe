# Project Trybetunes

# Contexto
Projeto 10 - Ciclo de Vida e React Router

## Técnologias usadas

Front-end:
> Desenvolvido usando: React, CSS3, HTML5, ES6, Jest

## Instalando Dependências

> Frontend
```bash
cd src/
npm install
``` 
## Executando aplicação

* Para rodar o front-end:

  ```
    cd src/ && npm start
  ```

## Executando Testes

* Para rodar todos os testes:

  ```
    npm test
  ```
