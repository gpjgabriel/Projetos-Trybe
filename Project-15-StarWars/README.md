## Técnologias usadas

Front-end:

## Instalando Dependências

> Frontend
```bash
cd src/
npm install
``` 
## Executando aplicação

* Para rodar o front-end:

  ```
    cd src/ && npm start
  ```

## Executando Testes

* Para rodar todos os testes:

  ```
    npm test
  ```
